# Web sound sources

These SFX files were imported from OpenGameArt packs and remapped for a more consistent in-game style.

## Pack sources
- 50 RPG sound effects (Kenney, CC0): https://opengameart.org/content/50-rpg-sound-effects
- Sound Pack 04 (rubberduck, CC0): https://opengameart.org/content/soundpack-04
- 75 CC0 breaking / falling / hit sfx (rubberduck, CC0): https://opengameart.org/content/75-cc0-breaking-falling-hit-sfx
